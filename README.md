# Flutter Text Helpers

A collection of widgets which simplifies the usage of `Text()`

## Usage

A simple usage example:

```dart
import 'package:flutter_text_helpers/flutter_text_helpers.dart';

main() {
  var awesome = new Awesome();
}
```
